# Olist Data Engineering Pipeline

Projeto de Engenharia de Dados ponta a ponta utilizando a base pública da Olist.

O objetivo é simular um pipeline próximo de um ambiente corporativo, passando pelas etapas de ingestão, transformação, modelagem analítica e visualização dos dados.

---

# 🏗️ Arquitetura

```text
RAW → STAGING → SILVER → GOLD → POWER BI
```

---

# 🛠️ Stack

* Python
* PostgreSQL
* SQL
* Pandas
* SQLAlchemy
* Power BI
* Git & GitHub

---

# ⚙️ O que já foi desenvolvido

* Ingestão automatizada de CSVs
* Estruturação das camadas Staging e Silver
* Tratamento e padronização dos dados
* Conversão de tipos e tratamento financeiro
* Organização do pipeline para modelagem analítica

---

# 📊 Próximos Passos

* Construção da camada Gold
* Modelagem dimensional (Star Schema)
* Criação do dashboard no Power BI
* Automatização do pipeline
